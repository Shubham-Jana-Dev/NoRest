Phase 1: Pattern Matching (LIKE)
1. Write a query to display all details of students whose student_name starts with the letter 'S'.

Ans. Select * from student where student_name like "S%";

2. Write a query to find all students whose phone number ends with the digit '3'.

Ans. Select * from student where phone like "%3";

3. Write a query to find all students whose name contains the letter 'a' anywhere.

Ans. SELECT * FROM student WHERE Student_name LIKE "%a%";

4. Write a query to find the total number of students currently stored in the student table.

Ans. SELECT count(*) FROM student;

5. Write a query to find the average Age of all students.

Ans. SELECT AVG(Age) from student;

6. Write a query to display each unique Age and count how many students belong to that age group.

Ans. Select Age, count(*) from student group by Age;

7. Write a query to display the Age groups that have more than 1 student.

Ans. Select Age from student group by Age having count(*) >1 ;

8. Write a query to find the complete details (all columns) of the student(s) with the minimum roll number using a subquery.

Ans. SELECT * FROM student where roll = (select Min(roll) from student);

9. Without using the LIMIT clause, write a subquery to find the second lowest roll number in the table.

Ans. SELECT min(roll) FROM student where roll > (select min(roll) from student);

10. Write a query to find all students who are older than the average age of the class.

Ans. SELECT student_name from student where Age > (Select avg(Age) from student);

11. Write a query to display the names of the top 2 oldest students.

Ans. SELECT student_name from student Order by Age Desc limit 2;

12. Write a query to find the roll number of the student with the 3rd highest roll number using LIMIT and OFFSET.

Ans. SELECT roll from student Order by roll desc limit 1 offset 2;

13. Write the command to completely remove the phone column from your student table.

Ans. Alter table student drop column phone;

14. What is the exact command you would write if you wanted to clear all rows from the orders table instantly and reset any auto-incrementing IDs, without deleting the table structure itself?

Ans. truncate orders;
